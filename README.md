# 라임포토랩 (Lime PhotoLab)

갤럭시 브라우저에서 바로 쓰는 단일 HTML 사진편집 앱.

- 구도 가이드 카메라 8종 (3분할·황금비·중앙·대각선·삼각형·수평선·프레임·대칭)
- 수동 보정 (밝기·대비·채도·온도·비네트) / 필터 9종 / 자르기·회전 / 텍스트 / 액자
- AI 보정 8종 (Gemini API — 어두운 사진 살리기, 저화질 복구, 피부 보정, 오래된 사진 복원, 눈 감은 사진 복구, 감성 햇살 추가, 배경 사람 제거, 망한 각도 수정)
